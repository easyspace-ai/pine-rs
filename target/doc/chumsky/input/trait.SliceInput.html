Expand description

Implemented by inputs from which slices can be extracted.

Note that there’s nothing to suggest that the slice needs to contain instances of the token type: it’s quite valid for an input to produce symbolic tokens, but slices corresponding to the original source code. However, it is conventional to do so.

type Slice: Clone

The unsized slice type of this input. For &str it’s &str, and for [&[T]] it will be &[T].
