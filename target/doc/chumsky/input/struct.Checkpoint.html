Struct Checkpoint 

Source 
pub struct Checkpoint<'src, 'parse, I: Input<'src>, C> { /* private fields */ }
Expand description

Represents a location in an input that can be rewound to.

Checkpoints can be created with InputRef::save and rewound to with InputRef::rewind.
