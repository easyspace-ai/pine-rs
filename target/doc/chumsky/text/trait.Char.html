Expand description

A trait implemented by textual character types (currently, u8 and char).

This trait is currently sealed to minimize the impact of breaking changes. If you find a type that you think should implement this trait, please open an issue/PR.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
